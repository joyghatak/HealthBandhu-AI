##  Overview

HealthBandhu is an AI-powered healthcare assistant that predicts possible medical conditions from patient-reported symptoms. It covers **754 disease classes** across **377 clinical symptom features**, trained on **246,945 patient records**.

##  System Architecture

```
Patient Symptoms (377 binary features)
         │
         ▼
┌─────────────────────────────────┐
│   EMERGENCY RULE ENGINE       │  Weighted critical symptom scoring
│  Threshold-based triage         │  CRITICAL / HIGH / NORMAL levels
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│   DEEP NEURAL NETWORK (DNN)   │  Primary — TensorFlow/Keras
│  377 → hidden layers → 754      │  Softmax probability over 754 diseases
└──────────────┬──────────────────┘
               │  (fallback if DNN unavailable)
               ▼
┌─────────────────────────────────┐
│   EXTRA TREES CLASSIFIER      │  Secondary — 100 estimators
│  754-class ensemble fallback    │  scikit-learn
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  EXPLAINABLE AI LAYER        │  Top-10 differential diagnoses
│  CLINICAL ASSISTANT          │  Evidence-based recommendations
│  REPORT GENERATOR            │  Downloadable clinical report (.txt)
└─────────────────────────────────┘
```

## 🔬 Technical Details

| Component | Details |
|-----------|---------|
| Primary Model | Deep Neural Network (.keras) — TensorFlow |
| Fallback Model | Extra Trees Classifier — scikit-learn |
| Input | 377-dimensional binary symptom vector |
| Output | Softmax probability over 754 disease classes |
| Emergency Engine | Weighted rule-based critical symptom scoring |
| Explainability | Top-K ranking + confidence calibration |
| Report | Downloadable structured .txt clinical report |
